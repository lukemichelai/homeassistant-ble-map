# homeassistant-ble-map

Home Assistant add-on repository for BLE Room Map.

## Add repository in Home Assistant

Use this URL in **Settings → Add-ons → Add-on Store → Repositories**:

`https://github.com/lukemichelai/homeassistant-ble-map`

Then install **BLE Room Map**.

The add-on UI is fully configurable:
- upload your own floor map image
- set map physical size
- place any scanner anchors by scanner_id
- define tracked-device aliases (no hardcoded scanner or device names)
